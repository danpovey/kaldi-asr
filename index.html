About this website

For now, this website is not the primary home of the Kaldi open-source speech recognition project; that remains at kaldi.sourceforge.net.

We have created this website in order to host Kaldi model builds, i.e. the outputs of running Kaldi recipes. We plan to post these here to the extent permitted by the copyrights on the original data (e.g. we won't be able to release things like data transcripts if these have been released under a restrictive license).

To browse the model builds that are available, please click on downloads. You will probably want to use wget to actually get the data- but, please, not from the top level!
