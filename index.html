About this website

Due to the long-running Sourceforge outage that occurred starting July 16th, we will now be designating this site as the primary home of the Kaldi project. Please now use kaldi-asr.org/doc as the canonical location of the Kaldi documentation (although it will also exist in the previous location at kaldi.sourceforge.net). To get the most recent code, you can use our github mirror https://github.com/kaldi-asr/kaldi. To checkout (i.e. clone in the git terminology) the most recent changes you can isse this command git clone https://github.com/kaldi-asr/kaldi or follow the github link and click "Download in zip" on the github page (right hand side of the web page)

We are currently assessing the feasibility of a transition to github.

This website exists to host the Kaldi documentation and also Kaldi model builds, i.e. the outputs of running Kaldi recipes. We plan to post these here to the extent permitted by the copyrights on the original data (e.g. we won't be able to release things like data transcripts if these have been released under a restrictive license).

To browse the model builds that are available, please click on downloads.

For instructions on how to upload your own model builds, see Uploading your builds.

This site is being hosted in the Google cloud, with the help of Google Cloud Platform credits (thanks, Google). If you have any suggestion of how to improve the site, please contact me. The source scripts for the site can be found at https://github.com/danpovey/kaldi-asr.git.
