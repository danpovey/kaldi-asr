Ways to talk/get help about Kaldi

You can sign for (generally reasonably low volume) Developers list (kaldi-developers@googlegroups.com) or for an User list (kaldi-help@googlegroups.com).

We generally strive to keep long discussion off-lists, so the traffic is not huge.

If you are in mainland China and can't access Google Groups directly, please ask jtrmal@gmail.com to add you to these lists (specify kaldi-help or kaldi-developers or both).

If you don't want to sign up, you can use the form bellow to read the kaldi-users list or use it to ask questions (about Kaldi, please).
