BUILDING SPEECH RECOGNITION SYSTEMS WITH THE KALDI TOOLKIT

Putty setup instruction

  • If putty is not installed on your computer, go to the download page and download the file putty.exe. You won't need the other programs.
  • Run putty.exe

You should be seeing a application windows looking like the one on the screenshot

  1. Choose the category Connection/Data
  2. Fill-out the Auto-login username field. Use ubuntu as the username

  1. Choose the category Connection/SSH/Auth
  2. Use the Browse button to select path to downloaded private key ibt-public.ppk You can download the key here.

  1. Click back on the category Connection
  2. Fill-out the Host Name (or IP address) field. Use the machine name you've been assigned.
      3. Optional:. Just to save you some time in case you will have to reconnect.
    1. Fill-out a name of the session. You can use something easy to remember or just the same name machine name as you've used above
    2. Click Save
  3. Click Open button.

  1. You will probably get asked if you trust the remote machine. Click Yes.

  1. Your screen should look similarly to his one.
Congratulations, your PuTTY works!
